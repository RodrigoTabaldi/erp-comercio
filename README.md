# ERP Comércio – Sistema de Gestão Comercial

Sistema ERP desenvolvido para simular o gerenciamento completo de pequenas empresas, incluindo controle de vendas, estoque e financeiro.

## Sobre o Projeto

O ERP Comércio foi criado com o objetivo de reproduzir um ambiente real de gestão empresarial, permitindo o controle integrado das principais áreas operacionais de um negócio.

O sistema oferece uma interface intuitiva e funcionalidades essenciais para tomada de decisões, como relatórios automáticos e organização eficiente dos dados.

## Funcionalidades

- Controle de estoque
- Gestão financeira
- Registro e acompanhamento de vendas
- Geração automática de relatórios
- Consulta e organização de dados
- Interface gráfica amigável

## Tecnologias Utilizadas

- Python
- Flet
- SQLite

## Banco de Dados

O sistema utiliza SQLite, garantindo:

- Armazenamento leve e eficiente  
- Facilidade de integração  
- Segurança dos dados  
- Geração de relatórios operacionais automatizados  

## Objetivo

Este projeto foi desenvolvido com fins educacionais, visando:

- Aplicar conceitos de desenvolvimento de software
- Simular sistemas utilizados no mercado
- Praticar integração entre interface, lógica e banco de dados
